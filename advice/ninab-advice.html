Hi my name is Nina and I have some advice for you!

Don't give up!

Learning to code is challenging, it is confusing and sometimes its really frustrating. But don't give up!! With perseverance it will be so rewarding when it all works and you know you yourself built that!

The more code you write, the more code will make sense. If you get stuck, break it down step by step and don't give up!! Before you know it you will be a coder!
